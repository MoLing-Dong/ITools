## 安装与使用

### 前提条件

本仓库大部分使用Python编写，期间可能调用其他软件服务譬如FFmpeg等

### 安装步骤

1. **克隆项目**：
   ```bash
   git clone https://github.com/MoLing-Dong/ITools.git
   cd ITools
   ```

2. **安装依赖**：
   ```bash
   pip install
   ```
